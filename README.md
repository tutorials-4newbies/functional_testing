## Functional Testing in python

## Setup

create a virtualenv
linux 
```bash

python -m venv venv
source testing_venv/bin/activate

```

windows

```
python -m venv venv
testing_venv\Scripts\activate
```

After that
```bash
pip install -r requirements.txt
```
